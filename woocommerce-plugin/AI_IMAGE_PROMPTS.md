## 🎯 Tips for Best Results

### **For Icons (256x256):**
1. **Keep it simple** - Should be recognizable at 128px and even 64px
2. **High contrast** - Use bold colors and clear shapes
3. **No text** - Icons should work without words
4. **Centered** - Main element should be perfectly centered
5. **Flat or minimal 3D** - Avoid complex shadows or gradients

### **For Banners:**
1. **Leave text space** - Right 50-60% should be relatively clean
2. **Visual on left** - Main icon/graphic on left third
3. **Professional colors** - Purple, blue, teal (trust and tech)
4. **Subtle patterns** - Background patterns should be faint, not distracting
5. **High resolution** - Request high-quality output for sharp display

## 🔄 After Generation

### **Icon Post-Processing:**

1. **Resize for WordPress.org:**
   ```
   256x256 → icon-256x256.png
   128x128 → icon-128x128.png (downscale from 256)
   ```

2. **Optimize file size:**
   - Use TinyPNG.com or similar
   - Target: <50KB for icons

### **Banner Post-Processing:**

1. **Create both sizes:**
   ```
   1544x500 → banner-1544x500.png (retina)
   772x250 → banner-772x250.png (standard, downscale from retina)
   ```

2. **Add text overlay** (use Canva or Photoshop):
   - Plugin name: "AI Fraud Detection"
   - Tagline: "Protect Your Store with Machine Learning"
   - Stat: "85%+ Accuracy • Real-Time Detection"

3. **Optimize:**
   - Target: <200KB for banners
   - Use JPEG for photos, PNG for graphics

## 🆘 Troubleshooting

**If AI refuses to generate:**
- Remove "plugin" or "WordPress" from prompt
- Use generic terms: "app icon", "software banner"
- Focus on abstract concepts: "security symbol", "tech illustration"

**If quality is low:**
- Add: "high quality, 4K, professional, sharp"
- Specify: "vector style" or "clean digital art"
- Avoid: "photorealistic" (better for icons to be illustrated)

**If colors are wrong:**
- Add hex codes directly: "#764ba2 to #667eea gradient"
- Specify: "exact colors, no color variation"
